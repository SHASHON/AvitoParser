Разработанный Авито парсер представляет собой инструмент для автоматизированного сбора данных с онлайн-площадки Avito. Для парсинга используется библиотека Selenium, которая позволяет имитировать действия пользователя в браузере, такие как нажатие кнопок, заполнение форм и скроллинг страницы.

Полученные данные сохраняются в базу данных SQLite3 в виде кэша. Также результаты последнего поискового запроса записываются в файл Excel с помощью библиотеки openpyxl, что обеспечивает удобный способ анализа и визуализации полученных данных.

Таким образом, были выполнены все основные задания, а также дополнительный пункт под звездочкой №4. При необходимости можно доделать все остальные.

!!!ПО было протестировано на операционных системах MacOS Sonoma, Windows 10. Для запуска необходимо иметь браузер Google Chrome.
